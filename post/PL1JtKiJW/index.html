    docker更换为国内镜像

    由于网络传输及限制原因，在国内访问Doker官方镜像慢如蜗牛，为了加快访问速度珍惜时间，将官方源配置为国内源，国内常用的有：网易、USTC、阿里云。

    我们这里改为家大业大的阿里源：

    pc@ubuntu:~$ sudo vim /etc/docker/daemon.json

    {
"registry-mirrors": ["https://b******.mirror.aliyuncs.com"]
}
//https://b******.mirror.aliyuncs.com	docker加速源，需要自己申请。
我的加速地址就不放公开博客里了
申请访问 https://www.aliyun.com 注册自己的帐号
然后点击控制台——>产品与服务——>容器镜像服务。

    pc@ubuntu:~$ systemctl daemon-reload
pc@ubuntu:~$ systemctl restart docker	#重启docker
